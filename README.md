# TF-M8X
ThunderFly Octa Multicopter

## Parameters
    * Length: 2050 mm
    * Width: 2050 mm
    * Operating Empty Weight: 6.4 kg
    * Maximum Take-off Weight: 12 kg
    * Motors: 8 x Tarot 330kv 4008
    * ESCs: 8 x X-Rotor 40A
    * Fly Time: ~ 20 minutes
